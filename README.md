# Ray Tracer in Modern C++

A physically based ray tracer implemented in C++ following the Ray Tracing in One Weekend series and extended with additional rendering features.

Features
+ Ray-sphere and quad intersection
+ Lambertian diffuse materials
+ Metallic reflection with controllable roughness
+ Dielectric materials with refraction
+ Motion blur using moving objects
+ Image texture mapping
+ Procedural Perlin noise textures
+ Depth of field and defocus blur
+ Gamma correction
+ Volumetric rendering (constant medium)
+ Bounding Volume Hierarchy (BVH) acceleration
+ Rendering Results
